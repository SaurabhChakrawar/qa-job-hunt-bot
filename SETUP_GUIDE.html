1 Install Prerequisites

# Python 3.9+ required
python --version

# Create virtual environment (recommended)
python -m venv qa-job-env
source qa-job-env/bin/activate  # Linux/Mac
# OR: qa-job-env\Scripts\activate  # Windows

# Install all dependencies
pip install -r requirements.txt

# Install Playwright browser
playwright install chromium
üí° Tip: Use a dedicated venv to avoid dependency conflicts.

3 Parse Your Resume

The bot uses Claude AI to understand your skills from your resume PDF.

# Replace with your actual resume path
python main.py --parse-resume /home/you/resume.pdf

# OR on Windows:
python main.py --parse-resume C:\Users\You\resume.pdf

This creates config/resume_profile.json with all your skills extracted. You can also edit this JSON manually to add skills that aren't on your resume yet.

üí° Tip: Check the generated resume_profile.json and add any missing tools or skills manually. The more complete your profile, the better the matching!

4 Test Your Setup

# Test email works first
python main.py --test-email

# Then run a full pipeline immediately
python main.py --run-now

Check your inbox! You should receive the first report. If not, check logs/ folder for errors.

5 Enable Daily Scheduler (9AM IST)

Option A: Crontab (Recommended ‚Äî Linux/Mac/WSL)

# Open crontab editor
crontab -e

# Add this line (3:30 AM UTC = 9:00 AM IST):
30 3 * * * cd /full/path/to/job-automation && /path/to/python main.py >> logs/cron.log 2>&1

# Example:
30 3 * * * cd /home/rahul/job-automation && /home/rahul/qa-job-env/bin/python main.py >> logs/cron.log 2>&1

Option B: Python Scheduler (Background Process)

# Run in background (keeps running even if you close terminal)
nohup python scheduler/scheduler.py > logs/scheduler.log 2>&1 &

# View logs
tail -f logs/scheduler.log

Option C: Windows Task Scheduler

# Create a .bat file: run_job_bot.bat
@echo off
cd C:\path\to\job-automation
C:\path\to\python.exe main.py >> logs\cron.log 2>&1

# Then add in Windows Task Scheduler:
# - Trigger: Daily at 9:00 AM
# - Action: Run run_job_bot.bat
